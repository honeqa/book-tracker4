# Book Tracker

## Автор
Умашев Арсланбек

## Описание программы
Book Tracker — это GUI-приложение на Python для учета прочитанных книг.

## Возможности
- Добавление книги (название, автор, жанр, страницы)
- Фильтрация по жанру
- Проверка корректности данных
- Сохранение и загрузка данных в JSON

## Запуск
```bash
python main.py
